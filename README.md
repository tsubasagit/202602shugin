このサイトの運営方針・情報の取り扱いです。

## このサイトについて・情報の取り扱い

- このサイトは、衆院選の仕組みと政策比較の理解を助ける「選挙勉強用ガイド」です。一般の方が選挙を学ぶ際の参考にしていただくことを目的としています。
- 定数（465議席、小選挙区289・比例176）、過半数233・3分の2は310など、衆院の制度上の数字は総務省・選挙制度に準拠しています。
- ニュースの日付・議席数・代表名・情勢はシミュレーションや想定を含み、公示後の実際の情勢と異なる場合があります。政党の役職・代表名は公開情報に基づきますが、選挙前後に変更される場合があります。
- 「参考記事」リンクは各メディアの政治面トップ等へ誘導しています。個別記事の存在・内容は保証していません。外部リンクは相手先の都合で変更・削除される場合があります。
- 投票の判断はご自身の責任で行ってください。実際の投票前に、総務省の公示情報・各政党の公式サイト・マニフェスト・信頼できるニュースで必ずご確認ください。

## 開発者向け

### 記事・動画の管理

記事・動画は別ファイルで管理しています。追加・修正するときは次のJSONを編集してください。

- **`data/articles.json`** … メディアニュース記事（朝日・読売・NHK・毎日・日経・共同など）
- **`data/videos.json`** … 政党YouTube（`partyVideos`）と動画ニュース（`newsVideos`）

`js/load-data.js` がページ読み込み時にこれらを取得し、ニュース・動画エリアに描画します。

### アクセス計測（Google Analytics 4）

`index.html` の `<head>` 内にある GA4 のスクリプトで、`G-XXXXXXXXXX` を実際の**測定ID**に差し替えてください。  
[Google アナリティクス](https://analytics.google.com/) でプロパティを作成し、測定IDを取得できます。
